Case Study: Self-Balancing Electric Bike

Overview

This project is an inverted pendulum-based self-balancing electric vehicle designed for urban mobility. The system uses real-time sensor feedback and actuator control to maintain balance dynamically.

System Architecture

  • Controller: STM32F4 with FreeRTOS
  • Sensors: MPU6050 (IMU), Rotary Encoder (600 PPR)
  • Actuators: Linear actuators with BTS7960 drivers
  • Feedback: Tilt angle + speed-based control
  • Data Logging: SD card (SPI interface)

Control Strategy

  • Tilt angle determines actuator movement
  • Speed threshold ensures safety (actuators retract above limit)
  • Closed-loop feedback using IMU + encoder
  • Real-time task scheduling using FreeRTOS

Key Engineering Challenges

  • Actuator response delay affecting stability
  • Efficient power usage in continuous balancing
  • Sensor noise and filtering
  • Mechanical limitations of actuator system

Solutions & Improvements

  • Optimized control logic for smoother actuation
  • Implemented threshold-based safety logic
  • Improved sensor fusion for stability
  • Continuous testing and tuning on real terrain

Results

  • Achieved stable balancing under controlled conditions
  • Real-time response using embedded system
  • Successful integration of sensors, actuators, and control logic
