System Architecture

  • STM32F4 + FreeRTOS
  • MPU6050 (IMU) + Encoder

  • SD card logging (SPI)

Control Logic

  • Tilt angle → actuator correction
  • Speed threshold → safety override
  • Closed-loop feedback system
